# FEC

## Setup

1. Run `npm install`
2. Copy `configExample.js`, rename it to `config.js`, and enter your Github API token
3. Run `npm run live-server` to start Node
4. Run `npm run build` to start Webpack

## Tech Stack
* React
* Moment.js
* Styled Components
* Webpack / Babel
* Node / Express
* Jest / React Testing Library
* circleCI
