Git pull: takes data from the same branch from GitHub and adds it into the current branch

Git checkout <branch> : moves into diffrent branch
Git checkout -b <branch> : created a new branch and moves into it

Git merge <branch> : appends the commits from your current branch onto the specified branch, then deletes the current branch
Git fetch <branch> : prepend commits from specified branch onto your branch, does not modify specified branch
Git branch -d <branch> : deletes local branch

Git add : stages file to be commited
Git commit : Saves an instance of the files
Git push : sends all commits to GitHub
Git push -u origin <branch> : creates a new branch on github and sends all commits to that branch

Git branch : displays all local branches
Git branch -r : displays all GitHub branches
